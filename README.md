## 技术栈

vue-cli4 + vuex + vue-router + vant + axios + sass

## 项目运行

```
git clone https://github.com/Auroka/vue-h5-template.git

cd vue-h5-template

npm install(如果安装慢可以使用淘宝镜像【npm install -g cnpm --registry=https://registry.npm.taobao.org】-> cnpm install)

npm run serve

```
